# Todo 截止时间功能规格

日期：2026-04-24

## 目标

为 Todo 添加"截止时间"属性，支持日期或日期+时间，临近截止时间有视觉紧迫提示。

## 数据模型

```ts
interface Todo {
  id: string
  title: string
  completed: boolean
  deadline: string | null  // ISO 8601 datetime
}
```

- `null` 表示未设置
- 时间部分为 `00:00:00` → 纯日期模式（如 `"2026-04-25T00:00:00"`）
- 时间部分非 `00:00:00` → 日期+时间模式（如 `"2026-04-25T14:30:00"`）

## 紧迫判定

仅未完成 + 有 deadline 的 todo 参与判定。已完成项不显示任何紧迫提示。

### 三档判定规则

| 档位 | 日期模式（00:00:00） | 时间模式（非00:00:00） | 视觉样式 |
|------|---------------------|----------------------|----------|
| 过期 | 当前日期 > 截止日期（即截止日期的第二天起） | 当前时间 > 截止时间 | 红色左边框 + 红色右端文字 |
| 紧急 | 截止日期 = 当前日期的前一天 | 截止时间在当天内 | 橙色左边框 + 灰色右端文字 |
| 提醒 | 距截止日期 > 1天且 ≤ 3天 | 距截止时间 > 当天且 ≤ 3天 | 灰色右端文字，无边框 |

### 右端文字格式

- 过期：`"已过期"`
- 紧急（日期模式）：`"明天"`
- 紧急（时间模式）：`"2小时内"` / `"30分钟内"`
- 提醒：`"3天内"` / `"2天内"`

## UI 变化

### 列表项

```
┌─────────────────────────────────────────┐
│▎ ☐ 购买生日礼物              3天内      │  ← 提醒：灰色文字
│▎ ☐ 提交报告                  明天       │  ← 紧急：橙色左边框+灰色文字
│▎ ☐ 开会                      2小时内    │  ← 紧急：橙色左边框+灰色文字
│▎ ☐ 交水电费                  已过期     │  ← 过期：红色左边框+红色文字
│  ☐ 整理书架                              │  ← 无deadline，无提示
│  ☑ 完成设计稿                             │  ← 已完成，无提示
└─────────────────────────────────────────┘
```

- 倒计时文字显示在列表项最右端
- 过期文字红色，紧急/提醒文字灰色
- 紧迫程度主要通过列表项整体样式体现：过期红色左边框，紧急橙色左边框，提醒无边框

### 右侧边栏

在现有 Title 和 Status 之间新增"截止时间"编辑区：

- 日期选择器（必选，选了日期才启用时间选择器）
- 时间选择器（可选，清空后存为 00:00:00 即纯日期模式）
- 清除按钮（清除整个 deadline）
- 截止时间显示格式：日期模式显示 `2026-04-25`，时间模式显示 `2026-04-25 14:30`

## 依赖

- shadcn Calendar + Popover + 原生 `<input type="time">`（参考 `src/components/shadcn-studio/date-picker/date-picker-10.tsx`）
- dayjs（日期格式化和计算）

- [ ] Todo 数据模型新增 deadline 字段

- [ ] 日期选择器和时间选择器分开，时间可清空

- [ ] 过期/紧急档显示彩色左边框
