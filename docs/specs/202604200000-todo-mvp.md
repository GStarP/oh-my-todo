# Todo 应用 MVP 规格

日期：2026-04-20

## 目标

极简 Todo Web 应用，只做核心流程。

## 数据模型

```ts
interface Todo {
  id: string
  title: string
  completed: boolean
}
```

只两个字段：标题、是否完成。不多加。

## 功能描述

### 1. 创建

- 顶部输入框 + 回车添加 todo

### 2. 列表

- 展示所有 todo
- 点击 checkbox 切换完成状态
- 完成项显示删除线

### 3. 编辑/删除（右侧边栏）

- 点击列表中的 todo，右侧展开边栏
- 边栏内可修改标题、删除 todo
- 点击空白处或关闭按钮收起边栏

### 4. 数据持久化

- localStorage 存取
- 存储层做抽象（Repository 模式），方便后续接入 DB

## 界面布局

```
┌──────────────────────────┬──────────┐
│  [输入框]                │          │
│  ☐ Todo 1               │  编辑区  │
│  ☑ Todo 2               │          │
│  ☐ Todo 3  ← 点击选中   │          │
│                          │          │
└──────────────────────────┴──────────┘
```

左侧列表，右侧边栏（选中时展开）。

## 验收标准

- [ ] 能增删改查 todo
- [ ] 完成状态切换正常
- [ ] 点击 todo 展开右侧编辑边栏
- [ ] 边栏内可修改标题、删除
- [ ] 刷新页面数据保留
- [ ] 存储层抽象，可替换实现

## 技术方案

- React 19 + Vite + TypeScript
- TailwindCSS
- localStorage 持久化（Repository 抽象层）

## 不做

- 分类/标签
- 筛选
- 子任务
- 拖拽排序
- 后端 API
- 用户认证
