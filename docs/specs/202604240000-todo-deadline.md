# Todo 截止时间功能规格

日期：2026-04-24

## 目标

为 Todo 添加"截止时间"属性，支持日期或日期+时间，临近截止时间有视觉紧迫提示。

## 数据模型

```ts
interface Todo {
  id: string
  title: string
  completed: boolean
  deadline: string | null
}
```

- `null` 表示未设置
- 时间部分为 `00:00:00` 表示纯日期模式
- 时间部分非 `00:00:00` 表示日期+时间模式

## 功能描述

### 紧迫判定

仅未完成且有 deadline 的 todo 参与判定。已完成项不显示紧迫提示。

| 档位 | 日期模式 | 时间模式 | 视觉样式 |
|------|----------|----------|----------|
| 过期 | 当前日期 > 截止日期 | 当前时间 > 截止时间 | 红色左边框 + 红色文字 |
| 紧急 | 截止日期为今天或明天 | 截止时间在今天内 | 橙色左边框 + 灰色文字 |
| 提醒 | 距截止日期 2-3 天 | 截止时间在未来 3 天内 | 黄色左边框 + 灰色文字 |

### 右端文字格式

- 过期：`过期`
- 紧急（日期模式）：`今天` / `明天`
- 紧急（时间模式）：`14:30`
- 提醒（日期模式）：`2天内` / `3天内`
- 提醒（时间模式）：`4月26日 14:30`
- 3 天之外（日期模式）：`4月28日`
- 3 天之外（时间模式）：`4月28日 14:30`

### 列表项

- 有 deadline 就显示右端文字
- 过期文字红色，其它为灰色
- 通过左边框颜色体现紧迫程度

### 右侧边栏

- 在 Title 和 Status 之间新增截止时间编辑区
- 日期选择器必选，时间选择器可选
- 清空时间后按纯日期模式保存
- 支持清除整个 deadline

## 验收标准

- [ ] Todo 数据模型新增 deadline 字段
- [ ] 右侧边栏可设置、修改、清除截止时间
- [ ] 日期选择器和时间选择器分开，时间可清空
- [ ] 列表项右端显示截止时间文字
- [ ] 过期、紧急、提醒三档判定正确
- [ ] 已完成项不显示紧迫提示
- [ ] localStorage 兼容旧数据

## 不做

- 浏览器通知
- 截止时间排序和筛选
- 重复截止时间
